Git（必须）

配置用户名、邮箱、生成 SSH 密钥

成功标志

且 C:/Users/hcx/.ssh 文件夹下生成 id_rsa.pub 文件

且将 id_rsa.pub 内容添加到 github.com

Vscode（非必须）

下载安装

使用 github 生成一个个人网站

创建仓库

进入创建仓库页面

填写仓库名，格式必须为<用户名>.github.io，然后点击Create repository

完成后访问<用户名>.github.io就是我们的主页

创建 Hexo 项目

安装 hexo-cli 脚手架，帮你创建 hexo 项目

阅读量统计

注册LeanCloud，进行邮箱、实名认证。

创建应用，选择免费配置即可

打开主题目录 themes\fluid下的 _config.yml 文件

修改 web_analytics -> enable: true

搜索 PV、UV，修改 statistics -> enable: true & source: "leancloud"，展示底部的 总访问量 和 总访问量
